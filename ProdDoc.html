What is IoT Core

 vitalpointz IoT Core is a cloud native server software that allows various IoT Devices to securely connect and transfer messages. Provides tools to consume the messages, process, and implement business logic on the messages sent by the IOT devices, transfer the messages or computed results to external third-party services. It also allows various users to be created and manage their own data and devices. It does all the above functions safely and securely, protecting itself (server software) as well as IoT devices.

 vitalpointz IOT Core runs IOT specific protocol services to interface IOT Devices with varying processing and power budgets.
