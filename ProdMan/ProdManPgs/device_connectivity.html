Device Connectivity

At transport level, Fog Nodes connect to vitalpointz IOT Core/Core Lite platform over SSL VPN. Every device that connects directly to the Cloud Platform connects over TLS connection using the Platform-issued certificates by default.

 

At the Service level, The Platform offers following interface services

  1. MQTT over TLS
  2. MQTT plaintext, Password protected
  3. MQTT over WebSocket
  4. HTTPS
  5. HTTP

 

Plain text interfaces such as HTTP and MQTT plaintext are provided to support the constrained devices. These interfaces need to be enabled specifically whereas all TLS based interface services are enabled by default.
