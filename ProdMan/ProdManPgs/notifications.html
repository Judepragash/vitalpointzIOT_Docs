Notifications

Notification service supports following notification delivery methods

  1. Email Notification: simple way of sending email to specified user whenever a condition or event occurs in cloud platform.
  2. In-App Notification: Whenever a user is logged in to the vitalpointz IoT Core/Core Lite platform, in-browser notification is delivered for the event.
  3. SMS Notification: In this method, notification is delivered to specified user’s mobile via text message. This requires integration with bulk sms providers and incurs additional cost. Reach out to support@vitalpointz.net if you need this service to be enabled.
