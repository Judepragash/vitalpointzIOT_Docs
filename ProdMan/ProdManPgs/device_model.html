vitalpointz Device Model

 In order to onboard and manage the IoT Devices, Following are the major phases the device should go through –   

  • Device registration
  • Device Onboarding
  • Device Removal

 

When a new device is registered, vitalpointz IoT Core / Core Lite does following step for the device.

  1. A unique Device Id and Device Topic is created
  2. Device certificate – Private Key and Certificate are created
  3. Based on the user selection, the Topic is whitelisted at the MQTT Broker.
