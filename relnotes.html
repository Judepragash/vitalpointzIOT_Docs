vitalpointz IoT Core Lite v1.0

 

Overview

 

Vitalpointz IOT Core Lite is a cloud native, built from ground up, microservice based IoT Platform. The software provides necessary security features to IoT Devices along with following features

 

  • IoT Device Registration, On-Boarding and life cycle management
  • Fog Node : virtualbox edition supported
  • User management , user privilege management
  • Rules Engine
  • Flow Processing using hosted NodeRED
  • Hosted NodeRED bundled with Vitalpointz custom nodes to receive/transmit data from/to Devices
  • Agents/SDKs: ESP01,NodeMCU and Simulator flows open sourced.
  • OutPort service to send data to external / third party services

About vitalpointz,Inc

vitalpointz is a software maker that specializes on developing and offering IOT platform softwares & services. IOT Devices and sensors can be registered on the IOT Platform and on-boarded securely. IOT devices can send or receive data securely, application can be built, Over-the Air software updates can be rolled out to the devices at scale. Vitalpointz IOT Platform also provides various interface methods to transfer the data to external third-party tools and platforms, if so desired. Vitalpointz IOT Platform also bundles various device agents.
